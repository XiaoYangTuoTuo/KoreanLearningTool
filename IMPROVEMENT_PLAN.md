# 功能完善计划 (IMPROVEMENT_PLAN.md)

为了全面提升 **Korean Learning Tool** 的用户体验，我们制定了以下详细的功能完善计划，涵盖打字练习、单词学习、语法研读和个人中心四个核心模块。

## 1. 点单练习 (Typing Practice) - 已部分优化

### ✅ 已完成
- [x] **批量练习支持**：允许用户选择 1/5/10/20/50 句进行批量练习。
- [x] **进度展示**：在练习过程中实时显示当前进度（如 3/10）。
- [x] **结果汇总**：练习结束后展示该批次的平均正确率、平均速度和总用时。
- [x] **计时修复**：修复了总用时无法正确累积的问题。
- [x] **输入体验优化**：优化了韩语输入法下的回车提交体验。

### 🚀 待开发 (Planned)
- [ ] **错题回顾模式 (Mistake Review)**：
    - 在结果页增加“查看错题”按钮。
    - 列出该批次中出错的句子，并高亮显示错误部分。
    - 提供“只练错题”的功能。
- [ ] **中途退出机制**：
    - 添加“放弃订单”按钮，允许用户中途结束练习并查看当前已完成部分的统计。
- [ ] **实时反馈增强**：
    - 打字过程中，正确/错误字符给予更明显的视觉反馈（如颜色微动效）。
    - 增加连击（Combo）计数特效，鼓励用户连续正确输入。

## 2. 每日单词 (Vocabulary)

### 🚀 待开发 (Planned)
- [ ] **学习进度持久化**：
    - **核心需求**：目前刷新页面后“已掌握”状态会丢失。
    - **方案**：使用 `localStorage` 或后端数据库记录用户的 `learnedWordIds`。
- [ ] **发音功能 (TTS)**：
    - 点击单词旁边的喇叭图标，播放韩语发音。
    - 使用 Web Speech API 或集成第三方 TTS 服务。
- [ ] **单词本收藏**：
    - 允许用户收藏生词到“生词本”。
    - 在个人中心提供“生词复习”入口。

## 3. 研读时光 (Grammar)

### 🚀 待开发 (Planned)
- [ ] **语法搜索功能**：
    - 在语法列表顶部添加搜索框，支持搜索语法点（如 "는/은", "过去时"）。
- [ ] **书签/笔记功能**：
    - 允许用户标记重点语法。
    - 支持对语法点添加简短的个人笔记。
- [ ] **关联练习跳转**：
    - 在语法详情页底部，提供跳转到相关“打字练习”句子的链接，实现学练结合。

## 4. 个人中心 (Profile)

### 🚀 待开发 (Planned)
- [ ] **数据可视化增强**：
    - 添加“最近7天学习时长”柱状图。
    - 添加“能力雷达图”（词汇量、打字速度、语法掌握度）。
- [ ] **成就系统**：
    - 设计徽章系统（如“早起鸟”、“打字机”、“百词斩”）。
    - 达成特定条件自动解锁徽章。
- [ ] **数据导出/导入**：
    - 支持导出学习记录为 JSON/CSV 备份。

## 5. 全局优化 (Global)

- [ ] **移动端适配优化**：进一步检查所有页面在手机端的布局表现。
- [ ] **深色模式 (Dark Mode)**：支持系统级或手动的深色模式切换。
- [ ] **性能优化**：对大型数据文件（如 `sentenceData.ts`）进行按需加载或分页处理，避免首屏加载过慢。

---

*此计划将根据用户反馈和开发进度动态调整。*
